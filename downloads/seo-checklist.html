1 Technical SEO Foundation

  • Connect custom domain
    Remove .myshopify.com and connect your branded domain with SSL certificate
  • Submit sitemap to Google Search Console
    Go to yourstore.com/sitemap.xml and submit to GSC for indexing
  • Verify site in Google Search Console
    Add DNS TXT record or HTML tag to verify ownership
  • Enable HTTPS redirect
    Settings > Domains > Check "Always redirect to primary domain with HTTPS"
  • Set up Google Analytics 4
    Install GA4 tracking code via Online Store > Preferences or Google channel
  • Configure robots.txt
    Review Settings > Files > robots.txt.liquid for proper crawl directives
  • Check mobile responsiveness
    Use Google's Mobile-Friendly Test tool on all key pages
  • Test Core Web Vitals
    Run PageSpeed Insights and aim for 90+ on mobile

2 Homepage SEO

  • Optimize homepage title tag
    Format: Brand Name | Primary Keyword | Secondary Benefit (under 60 chars)
  • Write compelling meta description
    Include primary keyword, value prop, and CTA (under 155 chars)
  • Add H1 heading with main keyword
    Only one H1 per page, should match search intent
  • Add descriptive homepage content
    Include 150-300 words of keyword-rich content below the fold
  • Internal link to key collections
    Add prominent links to your main product categories

3 Product Page SEO

  • Unique title tags for each product
    Format: Product Name - Key Feature | Brand (include target keyword)
  • Unique meta descriptions for products
    Highlight benefits, include keyword, add urgency or social proof
  • Write unique product descriptions
    300+ words, avoid manufacturer copy, focus on benefits and use cases
  • Add keyword-rich H1 for product title
    Include primary keyword naturally in product name
  • Optimize image alt text
    Describe product + color/variant + brand in each image alt tag
  • Add product reviews/ratings
    Enable reviews for rich snippet eligibility (star ratings in search)
  • Add FAQ section to products
    Answer common questions with FAQ schema markup
  • Optimize product URL handles
    Use keyword-rich, hyphenated URLs (products/blue-widget-set)
  • Add related products section
    Cross-link to related items for internal linking juice

4 Collection Page SEO

  • Optimize collection title tags
    Target category keywords: "Shop [Category] | Brand Name"
  • Write collection meta descriptions
    Include category keyword, number of products, key benefits
  • Add collection description (above products)
    150-300 words explaining the collection with target keywords
  • Optimize collection URL handles
    Use descriptive URLs: collections/mens-running-shoes
  • Add featured image with alt text
    Collection image should have keyword-rich alt text
  • Add bottom-of-page collection content
    Additional 200+ words after product grid for more keyword coverage

5 Image Optimization

  • Compress all images
    Use TinyPNG or Shopify's auto-compression; aim for under 200KB per image
  • Add descriptive alt text to all images
    Describe what's in the image + include relevant keywords naturally
  • Use descriptive file names
    Before upload: blue-running-shoes-nike.jpg (not IMG_12345.jpg)
  • Use WebP format where possible
    Shopify auto-converts, but upload high-quality originals
  • Implement lazy loading
    Most modern themes have this; verify in theme settings

6 Site Structure & Navigation

  • Create logical navigation hierarchy
    All products reachable within 3 clicks from homepage
  • Add breadcrumb navigation
    Enable breadcrumbs in theme for better UX and schema markup
  • Link to collections from main menu
    All main collections should be accessible from header navigation
  • Add footer links to key pages
    Include links to policies, about, contact, and popular collections
  • Create a sitemap page for users
    HTML sitemap page linking to all categories and key pages

7 Content & Blog SEO

  • Create blog with keyword-targeted posts
    Enable blog and plan content around buyer keywords
  • Write 1500+ word pillar content
    In-depth guides targeting main category keywords
  • Optimize blog post titles and meta
    Include target keyword in title, URL, and meta description
  • Add internal links in blog posts
    Link to relevant products and collections within content
  • Create how-to and buyer guide content
    "How to choose [product]" and "Best [product] for [use case]"
  • Publish consistently (2-4x/month minimum)
    Regular content signals freshness to Google

8 Schema & Structured Data

  • Verify Product schema is working
    Test with Google's Rich Results Test - should show price, availability
  • Add Organization schema
    Include logo, contact info, social profiles in site-wide schema
  • Enable Review schema
    Product reviews with proper schema for star ratings in SERPs
  • Add FAQ schema to relevant pages
    FAQ sections with proper JSON-LD markup
  • Add Breadcrumb schema
    Most themes include this; verify with structured data testing tool
  • Add LocalBusiness schema (if applicable)
    For stores with physical locations

9 Page Speed Optimization

  • Remove unused apps
    Audit and remove apps you don't use - they add code bloat
  • Minimize third-party scripts
    Each tracking pixel/chat widget slows your site
  • Use a fast, lightweight theme
    Dawn or OS 2.0 themes are optimized for speed
  • Limit homepage sections
    More sections = slower load; keep homepage focused
  • Use system fonts or limit custom fonts
    Each font file adds load time; max 2 font families

10 Local SEO (If Applicable)

  • Create Google Business Profile
    Claim and optimize your GMB listing with photos and info
  • Add NAP (Name, Address, Phone) consistently
    Same format across website, GMB, and all directories
  • Get listed in local directories
    Yelp, Yellow Pages, industry-specific directories
  • Encourage Google reviews
    Send follow-up emails asking for GMB reviews

11 Off-Page SEO & Link Building

  • Set up social profiles
    Create branded profiles on major platforms with link to site
  • Submit to relevant directories
    Industry directories, "best of" lists, local business listings
  • Reach out for product reviews
    Contact bloggers/influencers for product coverage and backlinks
  • Guest post on industry blogs
    Write valuable content for relevant sites with link back
  • Create shareable resources
    Infographics, tools, or guides that earn natural backlinks

12 Ongoing SEO Maintenance

  • Monitor Google Search Console weekly
    Check for crawl errors, indexing issues, and search performance
  • Track keyword rankings
    Use a rank tracker to monitor your target keywords monthly
  • Fix broken links quarterly
    Use Screaming Frog or similar to find and fix 404s
  • Update old content
    Refresh blog posts and product descriptions annually
  • Audit site speed quarterly
    Re-run PageSpeed Insights and fix new issues
  • Analyze competitor strategies
    Check what's working for competitors and adapt
Pro Tip: SEO is a marathon, not a sprint. Focus on completing all critical items first, then work through high priority items. Expect to see results in 3-6 months with consistent effort. Track your progress in Google Search Console and celebrate small wins along the way!
